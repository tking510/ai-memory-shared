# AI Memory Shared Repository (統一版)

## 🧠 目的
全AIエージェントが同じメモリ空間を共有、情報分散なし

## 📁 シンプル構造（TK修正版）
```
├── current-tasks.md      ← 全員の現在タスク
├── project-status.md     ← プロジェクト状況
├── daily-log.md          ← 日次作業ログ  
├── tools-urls.md         ← 重要URL・ツール集
├── decisions.md          ← 重要な決定事項
├── sync-log.md           ← 同期履歴
└── scripts/
```

## ⚡ ルール
1. **5分ごと監視**: 全エージェントが `git pull` で最新情報取得
2. **即時更新**: 作業完了時に即座 `git push`
3. **情報共有**: 分離なし、全員が同じ情報アクセス
4. **重複回避**: 1つのファイルに全情報集約

## 🎯 TK修正の利点
- ✅ 情報共有が自動
- ✅ 重複なし
- ✅ シンプル管理
- ✅ 全員同期
