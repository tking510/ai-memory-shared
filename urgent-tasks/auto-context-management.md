## 実装計画

### 1. 自動ファイル圧縮システム
```bash
# heartbeat毎実行: auto-context-manager.sh
- 120KB超過 → 警告、80KB目標でアーカイブ
- 150KB超過 → 緊急、60KB目標で大量アーカイブ  
- 170KB超過 → クリティカル、最小限ファイルのみ残し
```

### 2. 自動アーカイブ優先順位
1. **即座削除対象**: 
   - 完了済みタスクファイル
   - 古い研究文書
   - 重複ログファイル
2. **アーカイブ対象**:
   - 5KB以上の大きなファイル
   - 30日以上未更新ファイル
   - セットアップガイド系
3. **保護対象**:
   - AGENTS.md, TOOLS.md, HEARTBEAT.md, MEMORY.md
   - 当日のmemory/YYYY-MM-DD.md
   - shared-credentials.md

### 3. 自動セッション継承システム
```bash
# 170K超過時の自動処理
1. 重要な作業状況をAI Memory Sharedに保存
2. current-tasks.mdに進行中タスクを記録
3. 自動でmemory/session-backup.mdを作成
4. 新セッション開始時に自動復元
```

### 4. 監視システム
- **heartbeat毎**: コンテキストサイズチェック
- **120KB**: 予防的アーカイブ実行
- **150KB**: 緊急アーカイブ実行  
- **170KB**: セッション継承準備
